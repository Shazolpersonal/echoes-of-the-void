# Sound Assets

This directory contains audio files for the Echoes of the Void game.

## Required Files

Replace these placeholder files with actual MP3 audio:

- `wind.mp3` - Ambient wind/atmosphere sound
- `scream.mp3` - Horror scream effect
- `drip.mp3` - Water dripping sound
- `combat.mp3` - Combat/action sound effect

## Notes

- Keep audio files short (1-5 seconds) for sound effects
- Use MP3 format for browser compatibility
- Recommended: 128kbps or higher for quality
